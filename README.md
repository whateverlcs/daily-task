O Daily Task é um gerenciador de tarefas pessoal projetado para transformar a rotina em uma jornada de conquistas. Diferente de listas de tarefas comuns, este sistema utiliza elementos de RPG e Gamificação para incentivar a disciplina e o foco.

## Pré-Requisitos

1. .NET
2. SQL Server

## Como usar

1. Clone o repositório: `git clone https://github.com/whateverlcs/daily-task`.
2. Navegue até o diretório da solução: `cd daily-task`.
3. Realize o Build e rode a solução.

## Ferramentas Utilizadas

- .NET
- SQL Server
- Entity Framework Core
- Dapper
- FluentMigrator / FluentMigrator.Runner
- AutoMapper
- FluentValidation
- Caliburn
- Material Design
